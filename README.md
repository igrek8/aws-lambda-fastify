## Config

Create .env with the content below. Set `OMDB_ACCESS_TOKEN` for OMDB integration.

```
HOST=0.0.0.0
PORT=8080
NODE_ENV=production
OMDB_BASE_URL=http://www.omdbapi.com
OMDB_ACCESS_TOKEN=___YOUR_TOKEN___
LOG_LEVEL=debug
LRU_SIZE=25
LRU_MAX_AGE=432000
MOVIES_DIR=movies
```
